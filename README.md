# 📱 Google Play Store Analysis: Sustainability App Strategy

## 📝 Project Overview
This project was developed as part of the **Data Analysis & AI path at start2impact**.

The goal is to identify the most promising market niche for launching a new **sustainability-themed app**. By adopting a data-driven approach, I analyzed the Google Play Store dataset to find the perfect balance between user demand and market saturation.

## 🎯 Objectives
- **Data Cleaning:** Processing and formatting 10,000+ records (outlier handling, data type conversion, and cleaning categorical strings).
- **Exploratory Data Analysis (EDA):** Investigating distributions and correlations between Ratings, Reviews, Installs, and Price.
- **Market Strategy:** Identifying "Blue Oceans" (categories with high interest but manageable competition).
- **Hypothesis Testing:** Formulating and testing data-driven assumptions to help stakeholders make informed strategic decisions.

## 📊 Key Insights & Methodology
The analysis follows a structured workflow:
1. **Data Wrangling:** Transforming raw strings into numerical values (e.g., converting "1,000,000+" to integer format).
2. **Outlier Detection:** Identifying extreme values in pricing and installs to prevent statistical bias.
3. **Correlation Analysis:** Utilizing Heatmaps to visualize relationships between variables.
4. **Opportunity Scoring:** Developed a custom metric to rank categories based on Median Installs vs. App Density.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scipy
- **Environment:** Jupyter Notebook

## 📁 Repository Structure
- `data/`: Contains the raw dataset.
- `notebooks/`: The core Jupyter Notebook with step-by-step analysis and visualizations.
- `presentation/`: Interactive slides summarizing the strategy for stakeholders.
